# 밤의 목장 프리미엄 (v3.0) — 설계 스펙

날짜: 2026-07-07 · 상태: 사용자 승인 (시연 범위 "둘 다" 확정)
목표: 2026-07-08 시연회에서 (A) 발표 슬라이드 + (B) 앱 내 목업 데모로 "누가·언제·왜 결제하나"를 완결되게 보여준다. 실결제(PG) 연동은 범위 외.

## 1. 가치 제안

한 문장: **"커피 반 잔 값으로, 당신의 모든 밤이 한 권의 책이 됩니다."**

- 가격: ₩1,900/월, 연간 ₩19,000 (2달 무료)
- 파는 것: 기록의 소유권과 되돌아봄. 묵상 플로우 자체는 절대 유료화하지 않는다 (앱 정체성 "쉼" + 목회적 관점).

## 2. 무료/프리미엄 경계

| 기능 | 무료 (영구) | 프리미엄 |
|---|---|---|
| 묵상 플로우 4단계 | 전체 | 전체 |
| 말씀 보관함 | 무제한 | 무제한 |
| 스트릭·체크인 | O | O |

| 감정 통계 | 주간 차트 | 월간·연간 리포트 |

| 양 꾸미기 | 기본 양 | 소품 (매월 1개 추가 = 해지 방어) |

## 3. 프리미엄 기능 5종 상세

1. **기록 평생 보관** — 무료는 최근 60일 롤링. 카피 어조: "지워져요"(위협) 금지 → "무료는 최근 60일을 함께해요. 프리미엄은 당신의 모든 밤을 평생 간직해요."
2. **"나의 신앙 일기장" PDF** — 표지 일러스트 + 날짜별 감정 이모지 + 그날의 말씀 + 감사일기. 1차 피드백 "수필로 모아보기" 흡수. 실물 전환 가능한 유일한 산출물 = 소비자 관점 결제 메리트 1위.
3. **월간·연간 리포트** — 차트가 아니라 "돌보심의 기록" 프레이밍. 문장 요약 중심 (예: "불안했던 7일 중 6일, 시편 말씀이 함께했어요"). 연말 "올해의 밤" 결산(Wrapped 스타일)은 공유·바이럴 겸용.
4. **묵상 캘린더 + 챌린지** — 잔디 달력, "사순절 40일" 등 시즌 챌린지, 완주 기념 카드.
5. **양 꾸미기** — 결제 동인 아닌 유지 동인으로 설계.

## 4. 결제 유도 3지점

1. **61일째** (첫 기록 만료 임박): "두 달 전 그 밤의 기도, 계속 간직할까요?" — 최강 트리거.
2. **스트릭 30일 축하**: "한 달의 기록을 책으로 엮어드릴까요?" + 샘플 PDF 1페이지 무료 미리보기.
3. **대시보드 월간 탭**: 블러 + 자물쇠 (미리보기 후 잠금).

각 넛지는 1회성 모달 (반복 노출 금지 — 감성 훼손 방지).

## 5. 시연회 산출물 (2026-07-08)

### A. 발표 슬라이드 (presentation.html 확장)
- 프리미엄 경계표 슬라이드
- "결제 순간" 스토리: 워킹맘 수진, 61일째 밤 → 넛지 → ₩1,900 → 12월 PDF 일기장 수령
- 수익 추정 유지 (MAU 1,000 × 5% × ₩1,900 = ₩95,000/월)

### B. 앱 내 목업 (index.html)
- 프리미엄 안내 모달 (경계표 + 가격 + CTA)
- 대시보드 월간 탭 블러+자물쇠 처리
- 샘플 PDF "나의 신앙 일기장" 생성 (실데이터 기반 1개월치)
- 결제는 미연동: CTA → "곧 만나요" 또는 devMode(비번 7501)로 잠금 해제 시연
- 시연 팁: 샘플 PDF 실물 인쇄해서 심사위원 손에 쥐여주기

## 7. 제약

- 새 파일 금지: 앱 기능 전부 index.html 안에 구현. PDF 생성도 외부 라이브러리 CDN 지양 → 브라우저 인쇄(`window.print` + print CSS) 기반 우선 검토.
- Vanilla JS 전용.
- 콘텐츠(넛지 카피·리포트 문구)는 배포 전 /god 목회적 검토 필수.

## 8. 추가 확정 (2026-07-07, 같은 날 2차 확장)

- **배경 테마 프리미엄화**: 기존 5종 테마(밤하늘/새벽빛/숲속/바다/노을) 중 밤하늘만 무료 유지, 나머지 4종은 프리미엄 전용. 설정 오버레이 배경도 선택한 테마에 실시간으로 반응(라이브 프리뷰).
- **보관함 로그인/프리미엄 안내 배너**: 보관함 상단에 상태별 배너로 로그인 메리트(클라우드 동기화·기기 간 이어보기)와 프리미엄 메리트(평생보관·PDF·리포트·테마)를 명확히 분리해서 전달. 로그인 메리트에는 프리미엄 내용을 절대 섞지 않는다.
- **프리미엄 자물쇠 접근 방식 변경**: 자물쇠 5탭 → 기존 devMode 비밀번호(7501) 모달 재사용. 이미 체험 중이면 비번 없이 즉시 해제.
