01 · Overview
서비스 개요
하루를 마무리하는 감정 기반 말씀 묵상 앱 — 크리스천의 일상에 조용히 스며드는 디지털 쉼터
Core Value Proposition
"지금 내 감정을 선택하면,
오늘 나에게 맞는 성경 말씀이 찾아온다"
복잡한 검색 없이 · 3번의 선택 · 30초 안에 묵상 완료
🎯 핵심 플로우

감정 선택 → 주제 → 필요
→ 맞춤 말씀 + 묵상 + 기도문
+ 감사 일기 한 줄
